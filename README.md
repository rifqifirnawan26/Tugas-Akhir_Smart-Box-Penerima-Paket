# Smart Box Penerima Paket

Prototipe Smart Box Penerima Paket berbasis Internet of Things (IoT) dan Machine Learning menggunakan ESP32-S3 CAM, Edge Impulse, Firebase Realtime Database, Telegram Bot, dan Flutter.

## Features

- Automatic package detection
- Package classification using Machine Learning
- Real-time monitoring with Firebase
- Telegram notification
- Mobile application (Flutter)
- Automatic box status monitoring

## Hardware

- ESP32-S3 CAM
- Ultrasonic Sensor HC-SR04
- Load Cell + HX711
- Relay
- Buzzer
- Power Supply

## Software

- Arduino IDE
- Flutter
- Firebase Realtime Database
- Edge Impulse
- Telegram Bot API
